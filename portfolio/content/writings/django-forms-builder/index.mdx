# django-forms-builder (my first PR)

This is where my open-source story actually starts. I was using
django-forms-builder, a package that lets you build forms right in the Django
admin, for a college project. While using it I ran into a few shortcomings and
bugs, and instead of just working around them locally, I decided to try fixing
them at the source and opening a pull request.

The PR never got merged, and the project is not maintained anymore, so nothing
came of it in the end. But that is not really the point. It was the first time I
forked someone else's project, read their code, made a fix, and put it up for the
world to look at, and doing that in my early days taught me a lot about how open
source actually works.

## How you can use it

It is a Django package for building forms in the admin. My fork has the changes I
made back then. If you want to see where I started, the
[repo](https://github.com/krushiraj/django-forms-builder) is here.
