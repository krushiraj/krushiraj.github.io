# Match 4 Cards

A browser **card game** I built for fun with **React** and **Firebase** — a
light, quick game to play with friends without installing anything. React drives
the interface and Firebase handles the realtime state behind it, so the game
stays in sync across players.

It's one of a few small games I've built purely because they were fun to make,
alongside my [Catan Map Generator](/writings/catan-map-generator/) and
[Scotland Yard Companion](/writings/scotland-yard-companion/).

## How you can use it

Open it in a browser and play a round with friends — it's designed to be pick-up,
no-setup fun. **[Play it here](https://match-4-cards.web.app).**
