# Teaching & Tutorial Repos

These come out of the [full-stack course and guest lectures I teach](/#experience) at Vasavi College of Engineering. I build small, focused repositories so students have **clean, runnable material to learn from and refer
back to**, not just slides, but working code they can clone, run, and tinker
with. Each one is scoped to a topic and written to be read in order.
