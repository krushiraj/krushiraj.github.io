# Self-Hosted Backup Tools

At some point I decided to pull my data out of the various cloud accounts I was
paying for and move it onto my own NAS at home. The exports you get from Google
and Snapchat are a mess though, thousands of files with odd names, split ZIPs,
and metadata scattered around. So while I was doing my own migration, I wrote a
few small tools to make it painless, and cleaned them up so anyone else running a
home server can use them too.

They are the kind of thing I reach for when I want to actually own my data
instead of renting space for it. Each one does one job well.
